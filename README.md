# fake-account-detector
Alat analisis berbasis aturan yang ringan untuk mengidentifikasi akun media sosial palsu/bot menggunakan data perilaku. Alat ini dikembangkan khusus untuk keperluan Modul Forensik Digital sebagai sarana demonstrasi teknik deteksi praktis dan audit data perilaku tanpa bergantung pada model Machine Learning (ML) atau API platform pihak ketiga.
